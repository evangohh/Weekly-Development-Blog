Events and Local Storage

Events:
In web development, events refer to occurrences or interactions on a webpage, such as a user clicking a button, moving the mouse, or submitting a form.

Event Listeners:
Event listeners are functions in JavaScript that wait for a specific event to occur and then execute a predefined set of instructions or code.
They are attached to HTML elements and listen for events such as clicks, keypresses, mouse movements, etc.

Event Objects:
Event objects in JavaScript represent the state of an event that occurs in the Document Object Model (DOM). When an event, such as a mouse click or keyboard input, takes place, an event object is automatically created and passed to the event handler function associated with that event.

Some keypoints of event objects:
Event objects have properties that provide information about the event, including `type`, `target`, `currentTarget` and more.
