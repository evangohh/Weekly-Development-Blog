Objects and DOM

JavaScript objects and the Document Object Model (DOM) are fundamental concepts in web development, particularly for client-side scripting.

JavaScript Objects:
An object is a complex data type that allows you to store and organize data using key-value pairs.

Objects have properties, which are the variables associated with the object.
These properties can be of any data type, including other objects.

Objects can also have methods, which are functions associated with the object.

Simple Code:
let person = { name: 'John', age: 25, address: { city: 'Example City', country: 'Example Country' } };

Document Object Model (DOM):
The DOM is a programming interface for web documents. It represents the structure of a document as a tree of objects, where each object corresponds to a part of the document (e.g., elements, attributes).

JavaScript can interact with the DOM to dynamically update and modify the content, structure, and style of a web page
