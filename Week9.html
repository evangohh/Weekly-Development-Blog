BootStrap

Bootstrap is a popular open-source front-end framework used for designing and developing responsive and mobile-first web applications. It provides a collection of HTML, CSS, and Javascript Functions.

Key Features:
Responsive Design: Bootstrap helps in creating web pages that adapt and look good on various devices and screen sizes.
Grid System:It includes a flexible grid system based on a 12-column layout, making it easy to create responsive and well-organized layouts.


Components:
Bootstrap offers a wide range of components that make it easier for developers to create responsive and visually appealing user interfaces.
Navbar: A responsive navigation bar that can include navigation links, buttons, and other elements.
Grid System: A 12 grid system responsible for creating responsive layouts.
Typography: Pre-styled text elements for headings, paragraphs, and more.
Buttons: Styled buttons with various sizes and colors.
Forms: Styled form elements like input fields, checkboxes, and radio buttons.
