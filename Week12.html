What is SQL (Structued Query Language)

SQL is the core of the relational database which is used for accessing and maaging the database.
This language is used to manupulate and retrieve data from a structured data format in the form of tables and hold relationships between those tables.

What is NoSQL:
NoSQL, provides a mechanism for storage and retrieval of unstrucuted data.
This type of database can handle humongous amounts of data and has a dynamic schema.Unlike traditional relational database management systems (RDBMS) that use SQL (Structured Query Language) for querying and manipulating data, NoSQL databases offer a more flexible and scalable approach to data storage and retrieval.
