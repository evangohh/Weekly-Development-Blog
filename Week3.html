Week 3

Learning forms and advanced CSS styling.

HTML Form Structure: <form>Encloses the entire form, defining the area where user input is collected.

Input Elements: <input> : Creates various input fields (text, password, checkbox, etc.), specifying the type of input with the type attribute.

Labeling Input: <label> :Associates a label with a form element, enhancing accessibility and user experience.

Simple Validations: Basic validation can be achieved using: 'required' attribute: Makes an input field mandatory. 'pattern' attribute: Specifies a regular expression for input pattern matching.

Flex Boxes: Flexbox is a CSS layout model for creating flexible, efficient designs.
